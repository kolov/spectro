# Spectrometer Box Design Tool

Interactive transmission grating spectrometer geometry visualization for building a spectrometer box.

**Live demo:** https://kolov.github.io/spectro/spectrometer.html

Adjustable parameters:
- Slit-to-grating distance
- Grating density (300-1800 lines/mm)
- Grating-to-sensor distance
- Wavelength range
- Grating tilt
- Diffraction order

Shows box dimensions, component positions, light paths, and sensor span for building.
